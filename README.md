## The Vision

Most developer portfolios are "dark mode with some cards and a centered H1." This is different. This is a gritty, editorial experience that leverages scroll-driven mechanics to tell a story. It’s meant to look like a high-end fashion magazine had a baby with a 90s indie zine.

I have a 6.2 CGPA and a laptop that is trying its best. Most developers hide their weaknesses; I built a 100vw section around mine.

## Features

<br />

<table>
<tr>
<td width="50%" valign="top">

**Brutalist Nav Transition**  
A global full-screen curtain drop that slam-shuts before teleporting you to the next section. Snappy [0.76, 0, 0.24, 1] easing.

**Filmstrip Scroll-Pinning**  
A horizontal filmstrip layout pinned to a vertical scroll track. Elements slide sideways while you scroll down.

</td>
<td width="50%" valign="top">

**Dynamic Projects Table**  
Custom tabular layout with real-time GitHub data. Hovering over a row pops out a high-res screenshot centered on the viewport.

**Performance Native**  
Images optimized via sharp (.webp). Total initial load is under 200KB. CSS Grid enforced pixel-perfect alignment.

</td>
</tr>
</table>

## Local Setup

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repo
git clone https://github.com/AryanSingh64/Portfolio.git
cd Portfolio

# Install dependencies
npm install

# Start the dev server
npm run dev
```

### Deployment

The site is built to be deployed on **Vercel**. 

```bash
git add .
git commit -m "Sealing the deal"
git push origin main
```
